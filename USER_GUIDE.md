# Руководство пользователя

## Пользователь

1. Войдите по имени и фамилии либо откройте вкладку входа по логину.
2. На странице «Создать заявку» укажите тему, приоритет, категорию и описание.
3. В разделе «Заявки» доступны только разрешенные вашей ролью заявки.
4. Во вкладке «Выполнение» можно изменить доступный статус, обязательно указав комментарий к решению.
5. Во вкладках «Файлы» и «История» доступны вложения и журнал изменений.
6. Новые события отображаются в разделе «Уведомления».

## Сервисный сотрудник

1. Используйте режим «Назначенные мне» для личной очереди.
2. В основной информации заявки меняйте исполнителя, категорию и приоритет.
3. Во вкладке «Выполнение» выберите новый статус и опишите выполненные действия.
4. Комментарий к решению сохраняется в истории вместе со сменой статуса.

## Администратор

1. В разделе «Пользователи» создавайте и открывайте учетные записи.
2. Можно менять логин, имя, фамилию, роль, отдел, активность и пароль.
3. Архивирование сохраняет историю пользователя. Окончательное удаление доступно только без связанных данных.
4. В разделе «Права» настраиваются роли.
5. В разделе «Настройки» управляются пользователи, права, категории, отделы и SLA.
6. Журнал действий фиксирует административные и рабочие операции.

## Временный пароль

После сброса пароля администратором пользователь получает временный пароль. После входа система направит его в профиль для обязательной смены пароля.
